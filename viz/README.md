# Визуализации алгоритмов

Интерактивные дашборды для подготовки к коллоквиуму по алгоритмам и СД.

## Как запустить

Каждая визуализация — обычная HTML-страница, которая загружает библиотеки через CDN.  
**Нужен только браузер и интернет** (для загрузки шрифтов и D3.js при первом открытии).

### Вариант 1 — открыть напрямую (достаточно для большинства случаев)

```bash
open viz/01-dfs/index.html        # macOS
xdg-open viz/01-dfs/index.html   # Linux
start viz/01-dfs/index.html       # Windows
```

### Вариант 2 — локальный сервер (рекомендуется, если браузер блокирует файлы)

С Python (встроен в macOS/Linux):

```bash
# из корня репозитория
python3 -m http.server 8080
# открыть в браузере: http://localhost:8080/viz/01-dfs/
```

С Node.js:

```bash
npx serve .
# открыть: http://localhost:3000/viz/01-dfs/
```

## Управление (общее для всех дашбордов)

| Клавиша / действие | Эффект |
|---|---|
| `Space` | Play / Пауза |
| `→` или `.` | Шаг вперёд |
| `←` или `,` | Шаг назад |
| `R` | Сброс |
| Клик на холст | Добавить вершину |
| Клик → клик | Провести ребро |
| ПКМ на вершину / ребро | Удалить |
| Перетащить вершину | Переместить |
